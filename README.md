# Iris Dataset Exploration and Visualization

## Objective
The objective of this task is to understand how to read, summarize, and visualize a dataset using Python libraries. The Iris dataset is used to practice data inspection and create basic visualizations.

## Dataset
The Iris dataset is a well-known dataset in data science that contains measurements of iris flowers. It includes the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

The dataset was loaded using the seaborn library.

## Tools and Libraries Used
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Approach

1. Loaded the Iris dataset using pandas and seaborn.
2. Examined the dataset structure using:
   - `.shape`
   - `.columns`
   - `.head()`
3. Performed data visualization using matplotlib and seaborn.

### Visualizations Created

- **Scatter Plot**  
  Used to analyze the relationship between petal length and petal width across different species.

- **Histogram**  
  Used to understand the distribution of sepal length values.

- **Box Plot**  
  Used to detect outliers and analyze the spread of petal length values.

## Results and Insights

- Petal length and petal width show a strong relationship between species.
- Setosa species is clearly separable from the other species.
- Some variation exists in sepal measurements, but no extreme outliers were observed.

## Conclusion
This task helped in understanding how to load datasets, inspect their structure, and create meaningful visualizations for exploratory data analysis (EDA).
